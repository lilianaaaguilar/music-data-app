The data used in this web app is from the CORGIS Dataset Project and can be found here. The data contain statistics about music, the artists, its attributes (tempo, key, mode, etc.), and popularity ("hotttnesss").

The data includes songs from approximately 1978 to 2008. Many entries are missing or incomplete with some songs having no year listed, and many titles or release names are missing or appear as the number 0.

Each song, the dataset includes a lot of different measures that describe the musical characteristics and confidence levels in the data. For example, confidence values such as key_confidence or mode_confidence represent how certain the analysis system is about its predictions for musical key or mode. The mode value identifies whether the song is in a major or minor scale, which can influence the mood the song sounds.
